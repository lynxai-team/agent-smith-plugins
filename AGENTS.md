# Agent Smith plugins

This a Typescript mono repo project with several packages that are plugins for Agent Smith, a toolkit to manage AI agents

## Project context

To get the context of the project check `[project-path].agents/documentation/codebase-summary.md`. Each plugin has an `[plugin-path].agents/documentation/codebase-summary.md` file with detailled information for the package

## Documentation

The documentation files for the project are located at `agent-smith/docsite/public/doc`
